 -------------------------------------------------------------------------------------
 
 cd node-hbs-kmd-blk
 
 -------------------------------------------------------------------------------------

 -------------------------------------------------------------------------------------
 
 setup komodo rpc
 
 -------------------------------------------------------------------------------------
